# System Anomaly Detection using Machine Learning

## Overview

This project detects anomalies in system metrics using the Isolation Forest algorithm.

## Problem

Modern systems generate large volumes of logs and metrics. Detecting abnormal behavior manually is inefficient and often delayed.

## Solution

Built an anomaly detection model that identifies unusual patterns in:

* CPU Usage
* Memory Usage
* Response Time

##  Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## How it Works

* Simulated system metrics data
* Applied Isolation Forest (unsupervised learning)
* Detected anomalies in system behavior

## Real-world Applications

* Incident detection systems
* Fraud detection
* Monitoring and alerting systems
